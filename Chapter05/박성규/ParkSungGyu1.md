## 연산자 정리
1.값을 할당할 떄 사용하는 연산자: =
<br>
2.기본적인 덧셈, 뺼셈, 곱셈 나눗셈, 나머지를 계싼할 때 사용하는 연산자의 기호: +, -, *, /, %
<br>
3. ==와 !-의 차이는 ==은 같은지 비교, !=는 다른지 비교
<br>
4. < <=의 차이는 <는 미만, <=는 이하
5. ? : 로 표시하는 삼항 여산자의 ? : 뒤에 명시에 주는 값은 무엇을 의마하나요?
? 앞의 조건이 true일 경우 ?과 : 사이의 값을 리턴하고 false일 경우 : 뒤의 값을 리턴한다
(조건식) ? true일 경우 : false일 경우
